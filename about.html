About the project

This is the website for the Politecnico di Milano's GIS project (2019). This project revolves around the calculation of the Pedestrian Level Of Safety (PLOS) for a network of road in the 9th Municipality of Milan. The work was done in groups, and our group covered the area of Lancetti station, comprehensive of viale Vincenzo Lancetti, via dell'Aprica, viale Luigi Torelli, via Maloia, via Resegone, piazzale Costantino Nigra, viale Edoardo Jenner, via Edoardo Porro and via Teglio.

The PLOS is an index for evaluating walking conditions on road and street corridors in urban environment, so it is a useful parameter to measure comfort and safety of walkways.

The data were collected using the EpiCollect5 app and website through an on-field measurement.
